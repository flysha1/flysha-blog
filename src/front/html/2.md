# HTML 元信息

## 什么是 HTML 头部

HTML 头部包含 HTML <code style="color:#0085f2">\<head\></code> 元素的内容，与 <code style="color:#0085f2">\<body\></code> 元素内容不同，页面在浏览器加载后它的内容不会在浏览器中显示，它的作用是保存页面的一些元数据。上述示例的头部非常简短：

```html
<head>
  <meta charset="utf-8" />
  <title>我的测试页面</title>
</head>
```

<code style="color:#0085f2">\<title\></code> 元素也可以为文档添加标题，但和 <code style="color:#0085f2">\<h1\></code> 元素并不相同。

- <code style="color:#0085f2">\<h1\></code> 元素在页面加载完毕时显示在页面中，通常只出现一次，用来标记页面内容的标题（故事名称、新闻摘要等等）。
- <code style="color:#0085f2">\<title\></code> 元素是一项元数据，用于表示整个 HTML 文档的标题（而不是文档内容）。

## 元数据：\<meta\>元素

元数据就是描述数据的数据，而 HTML 有一个“官方的”方式来为一个文档添加元数据—— <code style="color:#0085f2">\<meta\></code> 元素。当然，其他在这篇文章中提到的东西也可以被当作元数据。有很多不同种类的 <code style="color:#0085f2">\<meta\></code> 元素可以被包含进你的页面的 <code style="color:#0085f2">\<head\></code> 元素，但是现在我们还不会尝试去解释所有类型，这只会引起混乱。我们会解释一些你常会看到的类型，先让你有个概念。
